# 概要
2つ目に作ったDjangoアプリ。Todoリストです。

# 起動方法
以下の2つを実行して下さい。

## マイグレーション実行

python manage.py migrate

## サーバー起動

python manage.py runserver 

# 使い方

## Todoリスト一覧ページにアクセス

http://localhost:8000/list/

## todoの作成
「新規作成」ボタンを押下し、todo作成ページから作成して下さい。

## todoの編集、削除、詳細表示
todoに表示される「編集画面へ」、「削除画面へ」、「詳細画面へ」のボタンで、  
編集、削除、詳細表示を行う画面へ遷移出来ます。
